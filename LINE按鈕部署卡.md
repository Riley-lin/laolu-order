## UAT 驗收清單

- [ ] **新單卡片**：下一張測試單 → 你的 LINE 收到的是**按鈕卡片**（品項＋金額＋✅25/30/35 分＋❌取消），不再是純文字
- [ ] **一鍵接單**：按「✅ 30分」→ 回「已接單 #XXX，取餐時間 XX:XX」＋一張「🔥 製作中」卡片（帶完成/取消鈕）
- [ ] **客人端連動**：綁定過的測試單，接單瞬間客人 LINE 收到取餐時間通知（跟以前老闆在後台按的效果一樣）
- [ ] **看單台同步**：這張單在看單台自動跳到「🔥 製作中」（Realtime）
- [ ] **防重複**：回到新單卡片再按一次「✅ 25分」→ 回「已經處理過囉」，時間**不會**被改成 25 分
- [ ] **完成**：製作中卡片按「🏁 完成」→ 回完成訊息；看單台變「✅ 已完成」
- [ ] **取消二段確認**：再下一單 → 按「❌ 取消這張單」→ 跳確認卡 → 按「留著不取消」→ 單沒事；再按取消 → 「確定取消」→ 看單台顯示已取消
- [ ] **回覆免額度**：以上按鈕回饋全走「回覆」（免費），只有新單卡片吃推播額度（跟以前一樣一單一則）
